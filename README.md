## Road Accident Analysis Dashboard


    One-Sentence Description:
    An interactive Microsoft Excel dashboard providing comprehensive analysis of road accident data, focusing on casualties, vehicle involvement, road conditions, and trends across different regions.

    
## Key Features & Analysis


    This dashboard utilizes Pivot Tables, Slicers, and dynamic charts to deliver immediate, data-driven insights.
    Casualty Breakdown: Clear metrics for Fatal, Serious, Slight, and Total Casualties (e.g., 417,883 Total Casualties)
    Contributory Factors: Detailed analysis of casualties segmented by Vehicle Type, Road Type, and Road Surface (e.g., wet vs. dry).
    Temporal Trends: A time-series chart showing Casualties by Year and month to easily identify trends, seasonality, and peak accident periods.
    Geographical Impact: Analysis of accident severity based on Location/Area (Urban vs. Rural) and Light Condition (Daylight vs. Dark).
    Interactive Filtering: The right-hand Filter Panel enables users to dynamically filter all visualizations by Accident Severity, Year, and Location Type.

    
 ## Accessing the Dashboard

 
 ## IMPORTANT: This is a large file (53 MB) and cannot be viewed directly in the GitHub browser. You must download the file to use it.

 
     Locate the File: Navigate to the main files list in this repository.
     Download: Click on the file named Road Accident Final Dashboard.xlsx (or similar) and choose Download.
     Software: Open the downloaded file in Microsoft Excel (Excel 2016 or newer recommended).

     
 ## How to Use

 
     Once opened in Excel, the dashboard is fully interactive:
     Use the Slicers in the Filter Panel (on the right) to instantly update all charts.
     Navigate between the summary and detail views using the tabs at the bottom (e.g., KPI, Data Analysis sheet, Road Type, etc.)
